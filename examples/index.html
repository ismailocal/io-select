AJAX ile Dinamik Arama

Aşağıdaki select, arama kutusuna yazdıkça AJAX ile backend'den ürünleri getirir (örnek API: dummyjson.com/products/search).

$('#ajaxSelect').ioSelect({
    placeholder: 'Ürün seçin',
    searchPlaceholder: 'Ürün ara...',
    noResultsText: 'Ürün bulunamadı',
    ajax: {
        url: 'https://dummyjson.com/products/search',
        method: 'GET',
        dataKey: 'products',
        queryParam: 'q'
    }
});
Seçili Ürün: 
null

AJAX ile Çoklu Seçim

AJAX ile Initial Selected

Bu örnekte AJAX ile çalışan select'te başlangıçta seçili değer ayarlanıyor. Ürün ID'si 1 olan "iPhone 9" başlangıçta seçili gelir.

$('#ajaxInitialSelect').ioSelect({
    placeholder: 'Ürün seçin',
    searchPlaceholder: 'Ürün ara...',
    noResultsText: 'Ürün bulunamadı',
    ajax: {
        url: 'https://dummyjson.com/products/search',
        method: 'GET',
        dataKey: 'products',
        queryParam: 'q'
    },
    initialSelected: {
        id: '1',
        name: 'iPhone 9'
    }
});
Seçili Ürün: 
null

AJAX ile Multiple Initial Selected

Bu örnekte AJAX ile çalışan multiple select'te başlangıçta birden fazla ürün seçili geliyor. iPhone 9 ve Samsung Galaxy başlangıçta seçili.

$('#ajaxMultiInitialSelect').ioSelect({
    placeholder: 'Ürün(ler) seçin',
    searchPlaceholder: 'Ürün ara...',
    noResultsText: 'Ürün bulunamadı',
    ajax: {
        url: 'https://dummyjson.com/products/search',
        method: 'GET',
        dataKey: 'products',
        queryParam: 'q'
    },
    initialSelected: [
        { id: '1', name: 'iPhone 9' },
        { id: '2', name: 'Samsung Galaxy' }
    ]
});
